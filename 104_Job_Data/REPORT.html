關鍵發現

在 2026 年 5 月 104 人力銀行上搜尋「暑期實習」，電視業納入分析的 23 筆職缺，全數不支薪—— 薪資欄位明確標示「論件計酬、薪資為零」，意即這些實習生的勞動，不換來任何金錢報酬。 23 筆分別來自TVBS 聯利媒體股份有限公司（12 筆）與八大電視股份有限公司（11 筆）。

另一端，本資料中月薪最高的職缺來自證券及期貨業： 「AI 量化實習生」，月薪 160,000 元。 職缺數最多的半導體製造業共 75 筆，其中逾七成月薪超過三萬元。

為呈現台灣暑期實習市場的薪資樣貌，本文透過爬取 104 人力銀行公開 API， 彙整 434 筆職缺名稱含「暑期」的實習機會，分析各產業的薪資揭露方式與職缺描述特徵。

資料來源與爬蟲設計

本次資料於 2026 年 5 月 24 日爬取自 104 人力銀行，以「暑期實習」為搜尋關鍵字， 透過平台公開 API 分兩階段取得職缺資料：

第一階段（搜尋列表）：呼叫搜尋 API，取得符合條件的職缺清單， 包含職缺編號、名稱、薪資摘要、產業別等基本資訊。
第二階段（職缺詳情）：以職缺 slug 逐筆呼叫 job detail API， 取得完整工作描述、條件要求等欄位。

遇到的困難

困難一：爬蟲中途電腦當機

第二階段逐筆抓取 detail API 期間，電腦意外當機導致程式中斷。 所幸程式設計了 Checkpoint 機制——每成功取得一筆 detail 後即即時將進度寫入暫存檔， 重啟後能從中斷點繼續，避免大量重工。

困難二：搜尋結果混入大量非暑期職缺

104 搜尋 API 的 kwop=7 參數並非嚴格限定職缺「名稱」比對， 只要工作描述中含有「暑期」字樣的職缺同樣會出現在結果中。 原始爬取共取得 2,090 筆，其中包含救生員、學年度正職工讀等大量非暑期實習職缺。

處理方式：過濾保留職缺名稱（job_name）含「暑期」字樣的職缺，共 465 筆； 再排除薪資格式不適用者（如論件計酬同時填有金額的職缺），最終納入分析 434 筆。 論件計酬且標有金額的職缺，其薪資以「完成件數」計算，實際收入取決於產出數量而非工時， 難以換算為等值的時薪或月薪，無法與其他薪資制度進行橫向比較，故予以排除。

圖三
各薪資組別職缺描述關鍵詞熱力圖（TF-IDF）
以 jieba 斷詞後對各薪資組別的合併職缺描述進行 TF-IDF 分析。 顏色越深代表該詞對該組越具有區別性；滑鼠懸停可查看確切數值。

TF-IDF（詞頻—逆文件頻率）衡量的是一個詞彙在特定文件群組中的「獨特重要性」： 若某詞在某一薪資組別的職缺描述中大量出現，卻在其他組別中罕見，則該詞對這個組別的 TF-IDF 值越高。 本分析將每個薪資組別的所有職缺描述合併為一份文件，再進行跨組比較。

從熱力圖中可以觀察到，各薪資組別的職缺描述在詞彙上存在一定差異。 值得注意的是，「學習」、「內容」、「能力」等詞在多個組別中均有較深的顏色， 或許反映了雇主對實習生的某些共同期待，不因薪資高低而有太大差別。 不支薪組的特色詞彙則偏向媒體製作相關技能（如「社群」、「剪輯」、「新聞」、「戲劇」）； 月薪 > 3萬組則較集中出現「分析」、「專案」、「開發」等工程研發導向詞彙， 且出現了「大四」一詞，或許反映科技業對高年級學生具備一定研究能力的期待。 這些詞彙差異或許反映了不同薪資條件下，雇主對暑期實習生的職能期待有所不同。

觀察與討論

根據這份 434 筆暑期實習職缺資料，可以初步觀察到幾個現象：

第一，電視業暑期實習或許存在不支薪的結構性傾向。 在本資料中，電視業 23 筆職缺全數標示不支薪，比例達 100%。 電視媒體產業對實習生的吸引力，是否讓求職者接受零薪酬的條件，值得進一步研究。 需注意，本資料僅為單日爬取結果，不代表整體市場長期樣貌。

第二，在本資料中，高薪實習高度集中在科技製造與金融領域。 月薪超過三萬元的職缺，主要來自半導體製造業、電腦及其週邊設備製造業、 IC 設計相關業、通訊機械器材相關業等科技業，以及人身保險業、證券及期貨業等金融業。 這些產業的高薪職缺，或許與業界對理工與金融人才的競爭性需求有一定關聯。

第三，職缺描述的文字差異，或許呈現了薪資與職能要求的關係。 TF-IDF 分析的初步觀察顯示，不支薪職缺的描述偏重媒體製作技能， 高薪月薪職缺的描述則偏重工程技術研發能力，兩者的語言生態明顯不同。
